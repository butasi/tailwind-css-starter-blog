> # `Overview`

The Arduino is an open-source platform for creating electronics projects that is built on simple-to-use hardware and software.
One thing unites all Arduino boards: they all run on microcontrollers.
For embedded applications, a microcontroller is a tiny computer having a CPU, memory, and other peripherals.

`Arduino Mega 2560`

> The Arduino Mega 2560 is a microcontroller board based on the ATmega2560. It has 54 digital input/output pins (of which 15 can be used as PWM outputs), 16 analog inputs, 4 UARTs (hardware serial ports), a 16 MHz crystal oscillator, a USB connection, a power jack, an ICSP header, and a reset button.
